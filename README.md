# 🔐 Cybersecurity Threat Analysis Dashboard

## Project Overview
Cybersecurity threats are rapidly increasing across industries and countries, causing major financial and operational damage.
This project focuses on transforming raw cybersecurity threat data into actionable insights using data analytics and visualization techniques.

## Tools Used

* SQL – Data analysis and querying
* Power BI – Dashboard creation and visualization
* DAX – KPI calculations
* Excel/CSV – Data source

## Project Objectives

* Analyze cyberattack trends over time
* Identify the most targeted countries and industries
* Find attack types causing the highest financial losses
* Analyze common security vulnerabilities
* Study the relationship between vulnerabilities and defense mechanisms
* Evaluate incident resolution performance
* Create an interactive dashboard for cybersecurity insights

## Dataset Information
The dataset contains cybersecurity incidents from 2015–2024 and includes:

* Country
* Year
* Attack Type
* Target Industry
* Financial Loss
* Affected Users
* Attack Source
* Vulnerability Type
* Defense Mechanism
* Resolution Time
* Risk Score
* Threat Level
* Attack Severity

## SQL Analysis Performed

The following questions were analyzed using SQL:

* Which countries face the most cyberattacks?

* Which attack types are most common?
* Which industries are most targeted?
* Which attacks cause the highest financial losses?
* What vulnerabilities are most frequently exploited?
* Which industries face the most critical attacks?
* Which attack combinations have the highest risk?
* Which industries take the longest to resolve incidents?
* Which defense mechanisms are used against specific vulnerabilities?

## Key Insights

* Cyberattacks have increased over the years.
* Some industries are targeted more frequently than others.
* Zero-day vulnerabilities represent significant security risks.
* Certain attack types cause higher financial losses.
* Antivirus and AI-based detection are among the most commonly used defense mechanisms.
* Some industries require more time to recover from cyber incidents.
* Geographic analysis helps identify cyberattack hotspots.
